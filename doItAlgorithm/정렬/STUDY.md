# 학습 내용

- 버블정렬
  - 두개씩 버블로 묶어서 swap하며 정렬함 -> 가장 맨 오른쪽부터 왼쪽으로 정렬됨
  - 시간복잡도 = O(n^2)
- 선택정렬
  - 최솟값 또는 최댓값을 찾고, 남은 정렬의 가장 앞에 있는 데이터와 swap
  - 시간복잡도 = O(n^2)
- 삽입정렬
  - 첫 2개부터 시작하여, 정렬된 데이터 범위에 정렬되지 않은 데이터를 적절한 위치에 삽입
  - 시간복잡도 = O(n^2)
- 퀵 정렬
  - pivot을 두고 left/right로 나누는 것을 반복하는 정렬
  - 재귀함수로 구현
  - 시간복잡도 = O(nlogn)

- Arrays.sort()
  - 평균 : O(nlog(n)) / 최악 : O(n^2)
- Collections.sort()
  - 평균, 최악 : O(nlog(n))
